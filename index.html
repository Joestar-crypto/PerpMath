How the valuation model works
1) Network size = sqrt(volume_7d * open_interest)
2) Network factor = (network_size / network_median)^0.5
3) Base value = (revenue_7d or fees_7d) * 52 * 15
4) Model FDV = base_value * network_factor
5) For launched tokens, ratio = model_fdv / current_FDV
6) For no-token protocols, future MC is shown as 35% of FDV.
